# CRUD 
#### My bicycle - CRUD, created on pure PHP
MVC Application to create, to read, to update, to delete posts.

### Usage:
* Twig
* MySQL



 
### Description

* Login / Sign up
* Pagination
* Add image to post

Main page shows all posts from DB (post included id, header, text, create date and time, author).

There are possibility to sort by date or header asc and desc.

User can to add an image to post also, the image is formatting proportional to 320px width or 240px height. 

Everyone can sign up by login and password and then use admin panel to create, update or delete own post.

There are super-admin. He can also create post and edit or delete all posts.
